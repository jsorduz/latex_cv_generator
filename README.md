# Latex CV Generator
The objectis of this repo is to create an interface that generates latex code to create your CV. You can upload a txt file with your information to use the graphic interface for small changes only.
This file needs to follow the following format:
```
<first_name>
<last_name>
<role>
<email>
<location>
<linkedin_label>
<linkedin_link>
<professional_profile>

SKILLS
_<skill_types>
<skill_1>
<skill_2>

EXPERIENCE
<date>
<role>
<company>
<location>
<description>
Habilidades: <skills>

<date>
<role>
<company>
<location>
<description>
Habilidades: <skills>

EDUCATION
<date>
<title>
<organization>
<location>
<description>

<date>

<date>
<title>
<organization>

<date>

<organization>

END
```
